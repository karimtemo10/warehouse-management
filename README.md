Warehouse Management System (CRUD)
A lightweight and efficient web-based application designed to manage warehouse inventory. This project demonstrates core frontend development skills including DOM manipulation, data persistence, and dynamic UI updates.

🚀 Features
Create: Add new products with details (Name, Price, Taxes, Discount, Category, Count.).

Read: Display inventory in a structured, real-time table.

Update: Easily modify existing product information.

Delete: Remove products from the system with a single click.

Search: Quick search functionality to find products by name or category.

Local Storage: Data is saved locally in the browser, so it's not lost on refresh.

🛠️ Tech Stack
HTML5
CSS3
JavaScript
